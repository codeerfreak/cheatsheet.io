HTML & CSS Cheat Sheets

HTML Table Reference

Tag Name Description
<table> table The wrapper element for all HTML tables.
<thead> table head The set of rows defining the column headers.
<tbody> table body This tag is used to group the body content in an
HTML table. Works like the <body> tag
<tr> table row The table row container.
<td> table data The set of rows containing actual table data.
<tfoot> table foot The set of rows defining the footer in a table.
